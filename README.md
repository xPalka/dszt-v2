# ds-zt-LGBMRegressor
LGBMRegressor model

You need `pip` and `python3`


1. Create venv
```bash
python -m venv venv
```

2. Activate venv
```bash
.\venv\Scripts\activate
```

3. Install dependencies in venv for jupiter
```bash
pip install -r requirements.txt
```

4. Run jupyter server
```bash
venv/Scripts/jupyter notebook
```
